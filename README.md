# Java Bootcamp Training - Class 1

Welcome to the Java Bootcamp! This repository contains exercises and examples from Class 1, covering fundamental Java programming concepts.

### Core Concepts
- **Arithmetic Operations**: Addition, subtraction, multiplication, division, and modulus
- **Comparison Operators**: Less than, greater than, equal to, not equal to
- **Logical Operators**: AND (&&), OR (||), NOT (!)
- **Increment & Decrement**: Pre/post increment and decrement operations

## üìÅ Files Overview

### 1. Calculator.java
A comprehensive demonstration of Java fundamentals including:
- **Arithmetic class**: Basic mathematical operations
- **Comparison class**: Relational operators demonstration
- **Logical class**: Boolean logic operations
- **incrementAndDecrement class**: Pre/post increment and decrement behavior
- **conditional class**: Age-based eligibility checks and switch-case statements
- **loops class**: Different loop types (for, while, do-while)

### 2. Task1.java
**ATM Withdrawal Simulation**
- Demonstrates nested conditional statements
- PIN verification logic
- Balance checking before withdrawal
- Real-world application of if-else statements

### 3. Task2.java
**Student Grading System**
- Grade calculation based on marks
- Uses if-else-if ladder structure
- Grading scale:
  - A: 90+
  - B: 80-89
  - C: 70-79
  - D: 60-69
  - Fail: Below 60
